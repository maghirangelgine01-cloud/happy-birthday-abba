# assets/

Place your photos and music file in this folder.

---

## 🎵 Music
- File name: **music.mp3**
- Any MP3 birthday/celebration song you like
- Free sources: https://pixabay.com/music/ (search "birthday")

---

## 📸 Photos (11 required)
| File        | Position              |
|-------------|-----------------------|
| photo1.jpg  | Top-left corner       |
| photo2.jpg  | Left side middle      |
| photo3.jpg  | Bottom-left corner    |
| photo4.jpg  | Top-right corner      |
| photo5.jpg  | Right side middle     |
| photo6.jpg  | Bottom-right corner   |
| photo7.jpg  | Bottom center         |
| photo8.jpg  | Left side lower-mid   |
| photo9.jpg  | Right side lower-mid  |
| photo10.jpg | Top center-left       |
| photo11.jpg | Top center-right      |

- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- Square photos look best (1:1 ratio)
- If a photo is missing, a soft pink placeholder will show instead

---

## 💌 Editing the Message
Open `script.js` and edit the `BIRTHDAY_MESSAGE` variable at the top of the file.
You can write as many lines as you want — the message box will scroll automatically.

```js
const BIRTHDAY_MESSAGE =
`Your long message goes here...
Line by line, it types out like a real letter. 💌`;
```

---

## 🚀 How to run
Just open `index.html` in any modern browser. No server needed!
